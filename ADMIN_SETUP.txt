SETUP INSTRUCTIONS:
------------------

Step 1: Install Dependencies
-----------------------------
Open terminal in project directory and run:
> npm install

This will install:
- bcryptjs (password hashing)
- chart.js & react-chartjs-2 (data visualization)
- express (backend server)

Step 2: Start the Admin API Server
----------------------------------
Open a NEW terminal and run:
> cd db
> node admin_server.js

OR simply double-click:
> start-admin-server.bat

The server will start on http://localhost:5000

Step 3: Start the React Development Server
------------------------------------------
In another terminal, run:
> npm start

This will start the main app on http://localhost:3000

Step 4: Access Admin Dashboard
------------------------------
Open your browser and navigate to:
> http://localhost:3000

Then modify the URL or create a link to open admin in new tab:
> window.open('http://localhost:3000', '_blank')

For development, you can directly access the admin by:
1. Temporarily modifying src/index.js to use admin-index.js
2. Or create a separate build for admin dashboard

TROUBLESHOOTING:
---------------
1. If admin server doesn't start:
   - Ensure port 5000 is not in use
   - Check if Node.js is installed (node --version)
   - Verify db/admin_server.js exists

2. If login fails:
   - Ensure admin server is running (check terminal)
   - Verify credentials (email: admin@ecommerce.com, password: admin123)
   - Check browser console for errors

3. If charts don't display:
   - Ensure db/full_database.json exists
   - Check browser console for Chart.js errors
   - Verify npm dependencies are installed
