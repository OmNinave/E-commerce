================================================================================
                          HOW TO RUN
================================================================================

STEP 1: Install Dependencies
----------------------------
> npm install

This installs:
- bcryptjs (password hashing)
- chart.js (charts library)
- react-chartjs-2 (React wrapper for Chart.js)
- express (backend server)
- cors (cross-origin requests)

STEP 2: Start Backend Server
----------------------------
Open Terminal 1:
> npm run admin-server

OR double-click:
> start-admin-server.bat

Server runs on: http://localhost:5000

STEP 3: Start React App
-----------------------
Open Terminal 2:
> npm start

App runs on: http://localhost:3000

STEP 4: Access Admin Dashboard
------------------------------
Open browser:
> http://localhost:3000/admin

OR from main site, open in new tab:
> <a href="/admin" target="_blank">Admin</a>

================================================================================
                        TROUBLESHOOTING
================================================================================

Problem: Server won't start
Solution:
  - Check if port 5000 is in use
  - Verify Node.js is installed (node --version)
  - Check for syntax errors in admin_server.js

Problem: Can't login
Solution:
  - Ensure admin server is running (check terminal)
  - Verify credentials (email: admin@ecommerce.com, password: admin123)
  - Check browser console for errors
  - Clear browser cache/localStorage

Problem: Charts not displaying
Solution:
  - Verify db/full_database.json exists
  - Check browser console for Chart.js errors
  - Ensure npm packages are installed
  - Try clearing browser cache

Problem: Session expires immediately
Solution:
  - Check system time is correct
  - Verify SESSION_TIMEOUT in admin_server.js
  - Check browser's cookie/localStorage settings

Problem: CORS errors
Solution:
  - Ensure frontend runs on http://localhost:3000
  - Check CORS configuration in admin_server.js
  - Verify both servers are running
