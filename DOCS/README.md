## üöÄ Quick Start Guide

### For Developers

1. **First Time Setup:**
   ```bash
   # Install dependencies
   npm install
   
   # Start backend server

   # Start frontend (in new terminal)
   npm start
   ```

   - Frontend: http://localhost:3000
   - Backend API: http://localhost:5000
