TROUBLESHOOTING:
----------------
Port 5000 in use?
‚Üí netstat -ano | findstr :5000
‚Üí taskkill /PID <PID> /F

Port 3000 in use?
‚Üí netstat -ano | findstr :3000
‚Üí taskkill /PID <PID> /F

Database not found?
‚Üí cd db
‚Üí node generate_unified_db.js

npm install fails?
‚Üí npm cache clean --force
‚Üí Delete node_modules and package-lock.json
‚Üí npm install
